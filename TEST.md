# 简易光鸭云 Chrome插件 - 测试清单

## 安装测试

1. 打开Chrome浏览器，访问 `chrome://extensions/`
2. 开启右上角"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择文件夹：`e:\trae\OpenList-main\guangya-chrome`
5. 确认插件已加载，无错误提示

## 功能测试

### 1. 设置页测试
- [ ] 点击插件图标 → 显示"请先登录"
- [ ] 点击"前往设置" → 打开设置页
- [ ] 点击"随机生成Device ID" → 生成32位数字
- [ ] 输入refresh_token → 点击"登录并验证"
- [ ] 验证成功 → 显示"已登录"

### 2. 文件列表测试
- [ ] 点击插件图标 → 显示文件列表
- [ ] 点击文件夹 → 进入子目录
- [ ] 点击面包屑 → 返回上级目录
- [ ] 点击刷新按钮 → 重新加载列表

### 3. 下载测试
- [ ] 点击文件 → 触发下载
- [ ] 查看Chrome下载列表 → 确认下载任务
- [ ] 下载完成 → 显示通知

### 4. 错误处理测试
- [ ] 未登录时 → 显示登录引导
- [ ] Token过期 → 自动刷新
- [ ] 网络错误 → 显示错误提示

## 调试方法

### 查看Service Worker日志
1. 访问 `chrome://extensions/`
2. 找到"简易光鸭云"插件
3. 点击"Service Worker"链接
4. 查看Console日志

### 查看Popup日志
1. 点击插件图标打开弹窗
2. 右键弹窗 → "检查"
3. 查看Console日志

### 查看Options日志
1. 打开设置页
2. 右键 → "检查"
3. 查看Console日志

## 常见问题

### Q: 提示"未登录"？
A: 请前往设置页输入refresh_token和device_id

### Q: Token刷新失败？
A: 检查refresh_token是否正确，可能已过期需要重新抓包

### Q: 文件列表为空？
A: 检查网络连接，查看Service Worker日志

### Q: 下载无响应？
A: 检查host_permissions是否包含API域名
